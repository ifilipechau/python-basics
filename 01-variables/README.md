# Variables - Exercises

This section contains exercises to help you understand how to use **variables** in Python.

## Learning Objectives

By completing these exercises, you will:
- Learn how to declare and assign values to variables
- Understand data types (int, float, str, bool)
- Use the `input()` function to get user data
- Perform basic arithmetic operations

## Exercises

| File | Description |
|------|-------------|
| `exercise_01_variables.py` | Declare and print variables |
| `exercise_02_variables_input.py` | Get user input and print it |
| `exercise_03_calculator.py` | Simple calculator using variables |
| `exercise_04_swap.py` | Swap two variables using a third variable |

## How to Run

Run any exercise using:
```bash
python exercise_01_variables.py
